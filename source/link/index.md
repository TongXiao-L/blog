## 友链互换
要添加友链，欢迎在评论区留言，以下是格式：
```yaml
- name: 博客名称
  link: 博客链接
  avatar: 博客头像链接
  desc: 博客描述
```

以下是本站的：
```yaml
- name: TongXiaoL的博客
  link: https://tongxiao-l.github.io/blog
  avatar: https://avatars.githubusercontent.com/u/114676932
  desc: Here is nothing...
```
